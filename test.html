Zendesk Ticket JSON Viewer

⚠️ Important: This file must be served from a local web server to work properly. Open a terminal/command prompt in this folder and run: python -m http.server 8000 (or python3 -m http.server 8000 on Mac/Linux), then visit http://localhost:8000/zendesk_ticket_viewer.html
How it works: This page uses your browser's existing Zendesk session cookies to fetch ticket data from https://gocareem.zendesk.com/api/v2/tickets/{id}.json. Make sure you're logged into Zendesk in this browser.
